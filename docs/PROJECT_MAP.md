# Engineering Management System v3

## LEGO v2 Modular Architecture

هذا المشروع هو نظام إدارة هندسية متكامل لشركة مقاولات، مبني على البنية المعيارية LEGO v2.

### الموديولات المخططة

| الموديول | الحالة | الوصف |
|----------|--------|-------|
| `engineering` | 🚧 قيد البناء | إدارة المشاريع الهندسية - المكتب الفني |
| `hr` | 📋 مخطط | إدارة الموارد البشرية |
| `finance` | 📋 مخطط | إدارة المالية والحسابات |
| `inventory` | 📋 مخطط | إدارة المخازن والمستودعات |

### البنية المعمارية

```
┌─────────────────────────────────────────────────────────────┐
│                    LEGO v2 Core Infrastructure               │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │ ModuleRegistry│  │ Connectors  │  │     EventBus        │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        ▼                     ▼                     ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│  engineering  │   │      hr       │   │    finance    │
│   (المكتب الفني)│   │  (الموارد البشرية)│   │   (المالية)   │
└───────────────┘   └───────────────┘   └───────────────┘
```

### التقنيات المستخدمة

- **Backend**: FastAPI + SQLModel + PostgreSQL
- **Architecture**: LEGO v2 (ModuleRegistry + Connectors + EventBus)
- **Pattern**: Modular Monolith with clear module boundaries

### المستخدمون الرئيسيون

- **المكتب الفني**: إدارة المشاريع، المستخلصات، المخططات، الجداول الزمنية
- **المديرون**: متابعة التقارير والمؤشرات
